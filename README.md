## WayMovies

Movie and TV Show Search App for iOS. Powered by TMDB.

Written in Swift 4.

+ Swipe to dismiss detail card.
+ Complex browse view layout with nested Scroll, Stack and Collection Views.
+ Active search

+ CosmosView
